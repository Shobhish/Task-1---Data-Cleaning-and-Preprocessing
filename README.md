# Task 1 - Data Cleaning and Preprocessing

## Objective
The objective of this task was to clean and preprocess the Netflix Movies and TV Shows dataset using Python (Pandas).

## Dataset
Netflix Movies and TV Shows Dataset

## Steps Performed
- Loaded the dataset using Pandas.
- Identified and handled missing values.
- Checked for duplicate records (no duplicate rows were found).
- Standardized text values by removing leading and trailing spaces.
- Converted the `date_added` column to datetime format.
- Verified that column headers were clean and uniform.
- Checked and verified data types.
- Exported the cleaned dataset as `cleaned_netflix_titles.csv`.

## Files Included
- `Task1.ipynb` – Jupyter Notebook containing the preprocessing code.
- `netflix_titles.csv` – Original dataset.
- `cleaned_netflix_titles.csv` – Cleaned dataset.

## Tools Used
- Python
- Pandas
- Jupyter Notebook

## Outcome
Successfully cleaned and preprocessed the dataset according to the given task instructions.
